   вҶ“
   Converts to вҶ’ InitiateTransferCommand

   вҶ“
   Check idempotency вҶ’ persistencePort.findByIdempotencyKey("abc-123")

## Components Summary

### Domain Layer (Pure Java - No Dependencies)
- **TransferDomain**: Immutable model with business rules
- **Use Case Interfaces**: What the application does
- **Output Port Interfaces**: What the application needs
- **Domain Services**: Business logic implementation

### Infrastructure Layer (Framework-Specific)
- **REST Controllers**: HTTP вҶ’ Domain
- **Persistence Adapters**: Domain вҶ’ Database
- **Workflow Adapters**: Domain вҶ’ Temporal
- **HTTP Adapters**: Domain вҶ’ External Services
- **Messaging Adapters**: Domain вҶ’ Kafka
